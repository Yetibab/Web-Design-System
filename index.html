Explore the system and discover how it can enhance your web design projects.

Colour

The Web Design System offers a palette of colours that are designed to work harmoniously together, ensuring a visually appealing and cohesive design.

Tips

  • Try to use complementary colours
  • Stick to a good colour palette

Buttons

Choosing the right buttons is mandatory when creating a website, making sure that the buttons match the context, matching the occasion.

Tips

  • Only use advanced types of buttons; ie. skeumorphic, neumorphic
  • Buttons should match the occasion, always make sure they match the website

Images

Almost all the time, images must match the context, as if not, this could cause viewer confusion, as well as a basic lack of clarity.

Tips

  • If using AI powered image generators, ensure that you give clear and consise prompts, leading to adequate pictures.
  • If searching for pictures online , make sure that you have checked licenses before use and that you take the best quality pictures.

Text

Text is a compulsory component seen almost anywhere and everywhere. Right structure is needed to confirm that websites are produced correctly

Tips

  • Make sure to use appropriate tags for headings, subheadings and paragraphs.
  • Text sizing should be taken into account.
